# TP-Link Tapo C310 카메라 RTSP 연동 가이드

Tapo C310 IP 카메라는 자체적으로 RTSP 미디어 서버 기능을 제공합니다.
집에 가셔서 카메라를 연동하실 때 아래 순서대로 설정을 진행해 주세요.

## 1. 카메라 계정(RTSP용) 생성하기
1. 스마트폰에서 **Tapo 앱**을 실행하고 카메라의 '라이브 뷰' 화면으로 들어갑니다.
2. 우측 상단의 **톱니바퀴 아이콘(설정)**을 누릅니다.
3. **고급 설정(Advanced Settings)** -> **카메라 계정(Camera Account)** 메뉴로 이동합니다.
4. 외부에서 스트림에 접속할 때 사용할 **아이디(Username)**와 **비밀번호(Password)**를 생성합니다. (기억해 두세요!)

## 2. 카메라 IP 주소 확인하기
1. Tapo 앱 설정 메뉴에서 **장치 정보(Device Info)**를 누릅니다.
2. 카메라의 **IP 주소** (예: `192.168.0.10`)를 확인합니다.
*(공유기 설정에서 이 IP를 고정 할당해 두시면 나중에 주소가 바뀌어 끊기는 일을 방지할 수 있습니다.)*

## 3. RTSP 주소 조합하기
위에서 얻은 정보를 바탕으로 아래 형식에 맞춰 주소를 만듭니다.

- **고화질 (1080p 이상)**: `rtsp://[아이디]:[비밀번호]@[카메라IP]:554/stream1`
- **저화질 (모바일용/360p)**: `rtsp://[아이디]:[비밀번호]@[카메라IP]:554/stream2`

> 예시: 아이디가 `admin`, 비밀번호가 `1234`, IP가 `192.168.0.100`인 경우
> 👉 `rtsp://admin:1234@192.168.0.100:554/stream1`

## 4. 서버(main.py) 코드에 적용하기
`main.py` 파일의 `RTSP_URL` 변수 또는 환경변수에 위에서 만든 주소를 입력하시면 곧바로 실시간 관제 스트리밍이 시작됩니다!
